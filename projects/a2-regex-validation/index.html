Validate Expresssion with RegEx











Name

Names can be "Md. Abdul Karim" , "Abdul Karim, Jr", "Jony" etc. Names cannot have numbers or other symbols like "Abdul1" "Abdul@karim"

Mobile Number

Bangladeshi 11 digit mobile number will start with "01" and will only have numbers. No space or "-" will be allowed here.

Post Code

Post codes are 4 digit numbers

Date

acceptable dates are of "5/5/2000" and "05/05/2000" format
